Sensibilite

Seuil actuel 150

Determine a quel point la roue doit accelerer ou freiner avant que le son change. Un chiffre bas rend l'app tres reactive mais peut se declencher pour de petits mouvements ; un chiffre haut demande une action plus franche. Ce reglage est sauvegarde automatiquement.

Marqueurs de capture (avance)

▾

A quoi ca sert : ces boutons ajoutent un repere dans le journal technique (onglet ci-dessous), pour marquer "a partir de maintenant, je fais telle action". Ca sert uniquement si tu dois analyser en detail comment la roue reagit — par exemple si le son se declenche mal et que tu veux comprendre pourquoi, ou pour verifier que le reglage de sensibilite est bien ajuste.

Comment les utiliser : juste avant de faire une action precise, appuie sur le bouton correspondant. Le bandeau ci-dessous confirme quel marqueur a ete pris en compte, meme si le journal technique defile trop vite pour le suivre a l'oeil.

Sequence de test recommandee

Pour verifier que tout fonctionne correctement, refais cet enchainement dans l'ordre, en appuyant sur le bouton correspondant juste avant chaque etape :

  1. Parking — allume la roue sans monter dessus
  2. Neutre — monte dessus, reste stable sans bouger
  3. Acceleration — accelerer franchement quelques metres
  4. Freinage — freine ou recule franchement
  5. Neutre — redescends a l'arret, roue toujours equilibree
  6. Parking — repose la roue, moteur coupe
